## 

- 首先，先激活环境
  
  ```python
  source activate target_env
  ```

- 然后输入下面的命令会在当前工作目录下生成一个environment.yml文件
  
  ```python
  conda env export > environment.yml
  ```

- 通过以下命令从该文件创建环境
  
  ```
  conda env create -f environment.yml
  ```

- 包管理
  
  ```
  conda list -n your_env_name
  ```

- 为指定环境安装某个包
  
  ```
  conda install -n env_name package_name
  ```
