## conda 环境移植

###1 首先，先激活环境
  
  ```python
  source activate target_env
  ```
  
###2 然后输入下面的命令会在当前工作目录下生成一个environment.yml文件
  
  ```python
  conda env export > environment.yml
  ```
  
###3 通过以下命令从该文件创建环境
  
  ```
  conda env create -f environment.yml
  ```
  
###4 包管理
  
  ```
  conda list -n your_env_name
  ```
  
###5 为指定环境安装某个包
  
  ```
  conda install -n env_name package_name
  ```
